# Dev Environment

Managed by [devbox](https://www.jetpack.io/devbox/).

## Setup

Install devbox, then:

    cd devbox && devbox install

## Adding packages

    devbox add PACKAGE_NAME

Search for packages: `devbox search PACKAGE_NAME`

Then commit the updated devbox.json.

## Languages included

- Python 3.12
- Node.js 20
- Go (latest)
- Ruby (latest)
- Java (JDK latest)
- Rust (via rustup)
